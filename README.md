# voting-app

## Run project

1. Create database 'voter' in MySQL db

2. Run .jar file of project:

    java -jar Voter.jar
    

## Install project

### Dependencies

1. MySQL

2. npm

3. Webpack 2


### Installing

1. Download UI dependencies

    mpn install
    
2. Build UI

    webpack
    
3. Build server side

    mvn clean install
    
4. Run project with maven

    mvn spring-boot:run
